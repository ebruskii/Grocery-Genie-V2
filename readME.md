# Grocery Genie v2

## To Run the Application Through Docker
```bash
docker build -t "api-server" .
```
```bash
docker-compose up
```

## To Run the Application Locally
```bash
npm i
```
```bash
npm start
```

## To Seed The File
```bash
npm run seed
```
## Server
Routes will be running on:
```
http://localhost:3000
```
